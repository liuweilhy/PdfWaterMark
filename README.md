# PdfWaterMark（Pdf文件批量添加水印、图章）

## 项目介绍
Pdf文件批量添加水印、图章的工具，特别适合批量图纸盖LOGO。

## 关于本工具
本工具基于.Net 4.6的WPF框架，使用Visual Studio 2017开发。未使用经典的MVVM构架，简化为MV构架。  
开发中使用了如下开源库：  
+ Pdfium.Net
+ WindowsAPICodePack
